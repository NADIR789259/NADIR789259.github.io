I'm Nadir Khan, a data analyst with a Diploma in Computer Engineering and a genuine obsession with understanding what data is trying to say. I don't just run scripts — I ask the right questions first, then let the data answer them.

My analytical journey started with building desktop applications in Python and gradually evolved into a full-stack data practice — from writing complex SQL queries and designing relational databases to building interactive Power BI dashboards that executives can actually use.

My strongest discipline is Exploratory Data Analysis (EDA) — I dig into raw datasets, clean them rigorously, and surface patterns, outliers, and trends that drive real decisions. From 150K ride bookings to e-commerce purchase behavior, every dataset has a story — and I know how to find it.

Currently building toward MNC-level Data Analyst roles while expanding my skill set into Generative AI, prompt engineering, and API integration as the next frontier.
